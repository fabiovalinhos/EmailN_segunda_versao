# EmailN (Segunda Versão)
## Domain

Um serviço para envio de e-mails em lote. O objetivo é evitar que o sistema cliente perca tempo enviando e-mails individualmente. 

Este projeto está sendo desenvolvido pela segunda vez para fins de reforço e melhoria.

#### Features

- Endpoint para envio dos e-mails em lote com os contatos.
- Endpoint para informar o status do lote enviado.
- Os e-mails enviados poderão ser personalizados por contato. Com isso, informações específicas do destinatário poderão ser incluídas no texto do e-mail.
